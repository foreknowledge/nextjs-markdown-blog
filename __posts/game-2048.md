# 2048 게임

### 실행 방법

https://enjoy-2048-game.netlify.app/ 에서 플레이

혹은

본 프로젝트를 클론 받아 아래 명령어로 앱 실행

```bash
> yarn
> yarn start
```

### 게임 방법

숫자 타일들을 합해 **2048 타일**을 만드는 게임입니다. <br />
공간이 부족해 타일이 더 이상 추가될 수 없으면 실패입니다.
